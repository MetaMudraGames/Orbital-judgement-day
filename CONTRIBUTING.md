# Contributing to MetaMudra Games
[span_64](start_span)We welcome educators, developers, and game designers who want to create innovative educational experiences.[span_64](end_span)

## How You Can Contribute
* **[span_65](start_span)Educational Content:** Propose new MetaDrama concepts or pedagogical reviews.[span_65](end_span)
* **[span_66](start_span)Safety:** Improvements to the safety framework or accessibility.[span_66](end_span)
* **[span_67](start_span)Technical:** Prompt optimization, documentation, or platform compatibility.[span_67](end_span)

## Rules for Contribution
1.  **Safety First:** All contributions must prioritize participant psychological safety. [span_68](start_span)No content designed purely to shock; no removal of safety mechanisms.[span_68](end_span)
2.  **[span_69](start_span)Educational Integrity:** Must serve genuine learning objectives with evidence-based approaches.[span_69](end_span)
3.  **[span_70](start_span)Platform Compliance:** No jailbreaking techniques or content that violates platform policies.[span_70](end_span)

## Pull Request Requirements
* **Description:** What does this contribution do?
* **Educational Rationale:** How does this serve learning goals?
* **[span_71](start_span)Safety Consideration:** Have you considered participant wellbeing?[span_71](end_span)
